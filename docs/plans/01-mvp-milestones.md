# MVP Milestones (Engineering)

## MVP definition

The MVP is “a usable Markdown vault in the browser” with links/backlinks, a local graph view, and semantic search + related notes.

## Milestone 1 — Markdown vault

- Create/edit/delete notes in Markdown.
- Render Markdown with wiki links, frontmatter, and tags.
- Basic navigation: note list, quick open/search box (keyword ok).

## Milestone 2 — Graph foundations

- Parse and persist `[[links]]` and tags.
- Backlinks panel.
- Local graph view (active note + neighbors).

## Milestone 3 — Semantic layer

- Embeddings generation and persistence.
- Semantic search.
- Related notes + link suggestions.

## Milestone 4 — AI augmentations (optional for MVP)

- Summarization action.
- Suggested tags.
- External AI integrations (OpenAI-compatible + Perplexity) with “save to vault”.
